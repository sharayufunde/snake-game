Classic Snake game built with Python and Tkinter.

## How to Play
1. Run `python snake_game.py`
2. Use arrow keys to control the snake
3. Eat red food to grow
4. Don't hit walls or yourself!

## Requirements
- Python 3.x
- Tkinter (usually included with Python)
